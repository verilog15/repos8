## pg_search

### New Features

- Added support for non-integer type keys for pg_search indexing. This includes - but is not limited to - strings and datetimes
- Added support for indexing UUID columns
- Upgraded the Tantivy version from 0.21.1 to 0.22.0

## ParadeDB

### New Features

- Added `pgvectorscale` to the ParadeDB Docker image, which is a complement to `pgvector` that significantly improve performance
- Upgraded `pgvector` from v0.7.0 to v0.7.2
