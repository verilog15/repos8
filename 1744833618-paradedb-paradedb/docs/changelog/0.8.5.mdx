## pg_search

### Bug Fixes

- Fixed an issue where index names that were over 52 characters in length were silently truncated
- Introduced new, refactored functions for highlighting and BM25 scoring that contain several robustness improvements

### New Features

- Added support for multi-language stemming
- Introduced partial BM25 indexing
