## pg_search

### Bug Fixes

- Fixed an issue where `DROP COLUMN` would cause BM25 indexing to crash.

## pg_lakehouse

### New Features

- Added support for running `EXPLAIN` and `EXPLAIN ANALYZE` queries over foreign tables.

### Bug Fixes

- Fixed a bug where users could not connect to Delta tables in Azure Blob Storage and ADLS Gen2.
- Fixed a bug where the Tokio reactor would crash following an error message.
