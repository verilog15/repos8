## pg_analytics

### Bug Fixes

- Fixed a bug where `timestamp` and `date` types were being stored incorrectly
- Fixed a bug where creating a heap table from a parquet table with `SELECT * INTO` or `CREATE ... AS SELECT *` was broken
- Disabled `DELETE` and `INSERT ... ON CONFLICT` clauses, which were not well supported

### Features

- Added support for the `time` type
